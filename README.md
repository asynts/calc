# calc
This is an interpreter for algebraic expressions. Example:

<pre><code><b>>>> 1</b>
1
<b>>>> (20 + 1) * 2</b>
42
<b>>>> foo</b>
unknown variable at :0
<b>>>> foo = 1</b>
1
<b>>>> foo + 2</b>
3
<b>>>> -(1 + 2)</b>
-3
<b>>>> help()</b>
name
name = expr

-expr
(expr)

expr + expr
expr - expr
expr * expr
expr / expr

help()
exit()
</pre></code>
